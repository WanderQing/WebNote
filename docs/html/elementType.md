# 元素类型

HTML有2种重要的元素类别，分别是**块级元素**和**内联元素**

## 块级元素（block）
相较于其前面的内容，块级元素会出现在下一行，其后的元素也会在下一行进行显示。块级元素不能嵌套在一个内联元素中，而可以嵌套在其他块级元素中。常见的有`<p>`、`<div>`。

## 内联元素（inline）
内联元素通常出现块级元素中，常见的有`<em>`、`<strong>`、`<a>`。
